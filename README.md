# MNIST 数字识别项目

这是一个使用 PyTorch 实现的简单深度学习模型，用于识别 MNIST 手写数字。

## 功能特性

1. 使用 PyTorch 实现简单的神经网络模型
2. 使用 MNIST 数据集进行训练和评估
3. 训练阶段：允许用户输入 epoch 数量来控制训练迭代次数
4. 评估模式：允许用户输入索引查看预测结果，并显示对应的文件路径和文件名

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py
```

程序会首先进入训练模式，输入 epoch 数量后开始训练。训练完成后可以进入评估模式，输入图片索引查看预测结果。
